# سندك - تطبيق مراقبة الصرع

## نظرة عامة
سندك هو تطبيق ويب متقدم لمراقبة ومنع نوبات الصرع باستخدام الذكاء الاصطناعي والأجهزة الطبية المتصلة.

## الميزات الرئيسية
- 🔬 **تحليل ذكي للبيانات**: استخدام OpenAI GPT-4 لتحليل نشاط المريض
- 📱 **واجهات ديناميكية**: تصميم متجاوب وجميل لجميع أنواع الأجهزة
- 🔔 **تنبيهات طوارئ ذكية**: إشعارات فورية للعائلة والأطباء
- 📍 **خرائط تفاعلية**: تتبع موقع المريض أثناء النوبات
- 🤖 **محاكاة الأجهزة**: وضع تجريبي للأجهزة الطبية
- 👨‍⚕️ **ربط بالأطباء**: نظام شامل للتواصل مع الأطباء
- 👨‍👩‍👧‍👦 **دعم العائلة**: إدارة جهات الاتصال الطارئة

## المتطلبات
- PHP 8.2 أو أحدث
- Composer
- Node.js و npm
- MySQL أو SQLite
- مفتاح OpenAI API

## التثبيت

### 1. تحميل المشروع
```bash
git clone <repository-url>
cd sanadak
```

### 2. تثبيت التبعيات
```bash
composer install
npm install
```

### 3. إعداد قاعدة البيانات
```bash
cp .env.example .env
php artisan key:generate
```

قم بتحديث ملف `.env` بالمعلومات الصحيحة لقاعدة البيانات.

### 4. إعداد OpenAI
احصل على مفتاح API من [OpenAI](https://platform.openai.com/api-keys) وأضفه إلى ملف `.env`:

```env
OPENAI_API_KEY=your_openai_api_key_here
OPENAI_MODEL=gpt-4
```

### 5. تشغيل المايقريشن والـ seeders
```bash
php artisan migrate
php artisan db:seed
```

### 6. بناء الأصول
```bash
npm run build
```

### 7. تشغيل الخادم
```bash
php artisan serve
```

## الأجهزة المدعومة
- 🧠 **EEG Emotiv EPOC X**: مراقبة نشاط الدماغ
- ❤️ **ECG Polar H10**: مراقبة معدل النبض وضغط الدم
- 💪 **EMG ESP32 MyoWare**: مراقبة توتر العضلات

## استخدام وضع المحاكاة
للاختبار بدون أجهزة حقيقية:

1. اذهب إلى قسم "الأجهزة" في لوحة التحكم
2. اضغط على "تشغيل وضع المحاكاة"
3. سيقوم النظام بإنشاء بيانات وهمية للأجهزة

## الذكاء الاصطناعي
يستخدم التطبيق OpenAI GPT-4 لـ:

- **تحليل النشاط**: تحديد ما يفعله المريض حالياً
- **تقييم المخاطر**: حساب احتمال حدوث نوبة
- **التوصيات الطبية**: نصائح مخصصة للمريض
- **البحث عن المستشفيات**: العثور على أقرب مراكز طبية

## API Endpoints
- `GET /ai-analysis`: تحليل ذكي للبيانات الحالية
- `POST /emergency/trigger`: تشغيل تنبيه طوارئ
- `GET /map`: عرض الخريطة التفاعلية
